deckgene 是一套**白标 AI 演示稿基础设施**。它把「给一段文字、一个链接或一份文档,自动生成一套排版精良、符合品牌调性的演示稿」这件事,封装成可以直接调用的能力——你既能在 deckgene 自己的控制台里用,也能通过 API / MCP 接入到你自己的产品里,几分钟就让用户拥有「AI 出片」功能,而不用自己从零搭一套幻灯片引擎。

## 从输入到成稿,发生了什么

1. **输入** — 粘贴文字、填一个网页链接,或上传文档。
2. **规划** — AI 先按叙事弧线(开场 → 论证 → 展示 → 收尾)规划整套结构,为每一页挑选合适的**页型**。
3. **成页** — 每一页都是**带 schema 的结构化页型**(标题、要点、数据、图表、图文、时间线……近百种),内容被严格约束,排版稳定不塌。
4. **套品牌** — 自动套用你的**品牌套件**:主色、字体、Logo,一键换肤,输出风格统一。
5. **交付** — 在线预览与全屏演示、发布成公开链接(`/d/你的slug`),或导出 PPTX / PDF。

## 为什么不是「又一个 AI PPT 工具」

- **白标 / Headless** — deckgene 是给你的产品用的基础设施,不是终端消费应用。品牌、入口、计费都可以是你的。
- **结构化而非自由排版** — 页型是写死 schema 的强约束,AI 只填内容不乱改版式,所以产出可预测、可批量。
- **可编辑** — 生成后有完整编辑器:逐页微调、AI 局部改写、撤销/重做、拖拽重排。
- **可编程** — REST API 与 MCP 双通道,AI agent 能直接建稿、加页、发布(见下一篇)。

## 适合谁

- 想在自己 SaaS 里加「一键生成演示稿 / 报告」的团队;
- 需要把结构化数据批量变成品牌化 deck 的场景(周报、销售材料、课程讲义);
- 想让 AI agent 自动产出演示稿的开发者。

把「做 PPT」变成一次 API 调用——这就是 deckgene。
